# Human-Following-Robots
It's a simple Arduino based human following car I made using an Arduino Uno, two ultrasonic sensors, 2 Bo motors, jumper wires, breadboard, power bank 
I had set it for  5cm distance object tracking.
if we can install a camera module in place of the ultrasonic sensors, this prototype can be scaled to include owner face identification and following and also commands can be given to it though hand gestures ( features of computer vision)
